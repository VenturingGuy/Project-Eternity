Hub Unit Editor

The hub unit editor is used to link Visual Novels to a Unit that will be used inside a Hub.

Visnual Novels:

This section contains the list of Visual Novels. The Unit only play one Visual Novel for a single interaction. The following interactions will play the following Visual Novel until it reach the last one.

After reaching the last Visual Novel it will keep repeating it. To have dynamic Visual Novel interaction it is recommended to have a script replace the Unit with another Hub Unit that has a separated set of Visual Novels.
