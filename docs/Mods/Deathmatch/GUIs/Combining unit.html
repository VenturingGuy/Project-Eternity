Combining Unit Editor

The transforming unit editor is used to link multiple units together and change from one unit to the other.

Combining Units:

This section contains the list of other Combining Units needed to transform into the Combined Unit. The first Unit in the list will be considered the leader, making the Combined Unit spawn in its place.

Original Unit:

This Unit is the Unit to use when not combined.

Combined Unit:

This Unit is the to one use after combination. It is only used if this Combining Unit is the leader of the combination. Can be left blank if it will never be used as a leader.
