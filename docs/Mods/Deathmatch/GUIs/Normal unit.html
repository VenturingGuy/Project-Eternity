Normal Unit Editor

 

Unit boosts:

Those are the basic statistics for the Unit.

  • Base HP: Short for Hit Point. How much damage the Unit can take before getting destroyed.
  • Base EN: Short for Energy. Energy is used by some attacks.
  • Base Armor: Armor is used to reduce damage received from enemies.
  • Base Mobility: Mobility is used to increase the chances of evading attacks.
  • Base Movement: Long for MV. How many tiles the Unit can move on the map.
  • Parts Slots: Maximum number of parts the unit can have.

Pilot whitelist:

Unit normally can't change the Character piloting it. You can make exceptions by adding Characters to the pilot whitelist.

Animations:

During a combat, a Unit will be able to react to the context and play the Animations linked to the corresponding behavior.

If no animation is set for a behavior the Default animation will be played instead.

Abilities:

 

Terrain:

Give a bonus to statistics during a combat. The bonus is based on the letter corresponding to the terrain the Unit is on.

The bonus ignore the current Movement type of the Unit and use the terrain over which the Unit is.

It will also halve the terrain movement cost if the rank for the terrain is S.

Air movement ignores terrain. If the unit's air rank is B or lower, movement range is halved.

Bonus chart for Mobility:

  • S: 20%
  • A: 10%
  • B: 0%
  • C: -15%
  • D: -30%

Bonus chart for Armor:

  • S: 20%
  • A: 10%
  • B: 0%
  • C: -10%
  • D: -20%

Movement types:

Defines where the Unit can move.

Size:

Give a bonus to offense and defense if the Size is bigger than the enemy.

If the Unit is smaller, there is no bonus or malus.

If the Unit is bigger, the bonus correspond to the value of the Unit minus the Enemy.

Ex. A 3L unit attacking a SS Unit will gain 50% damage bonus. The SS unit won't get any defense bonus.

Bonus chart:

  • SS: -20%
  • S: -10%
  • M: 0%
  • L: 10%
  • 2L: 20%
  • 3L: 30%
