Transforming Unit Editor

The transforming unit editor is used to link multiple units together and change from one unit to the other.

Units:

This section contains the list Units the Transforming Unit can change into.

Limitations:

  • Will requirement: The minimum Will the pilot of the Transforming Unit needs to be able to transform.
  • Turn limit: How long the Transforming Unit can stay transformed.
  • Skill requirement: Only allow transforming if a specific skill is active. If transformed and the skill is disabled, the transformation is cancelled.
  • Permanent transformation: Makes it impossible for the Transforming Unit to change back to its original Unit.
