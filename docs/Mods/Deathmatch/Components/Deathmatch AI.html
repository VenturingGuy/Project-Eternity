Deathmatch AI

Deathamtch AI scripts are AI scripts that can be used inside the Deathmatch Mod.

Get Map Variable:

Fields:

Variable Name: The name of the variable to get.

Requirement:

  • None

Description:

This script returns a number corresponding to the variable specified. Will crash if the variable is not set.

Set Map Variable:

Fields:

Variable Name: The name of the variable to get.

Sign Operator: The kind of operation to do with the variable. Can be equal, add, substract, multiply, divide or modulo.

Requirement:

  • None

Description:

This script will set or change the number of a variable.

Get Enemies:

Fields:

None

Requirement:

  • None

Description:

This script holds a list of Squad of every Squad from every Player that are of a different team than the AI's Squad.

Get Targets:

Fields:

Add Movement To Range: Consider that the Squad can move before attacking to extend the range of the Attack.

Requirement:

  • Leader Attack: Attack the Leader wants to use.

Description:

This script holds a list of Squad of every Squad from every Player that are of a different team than the AI's Squad and that can be attacked.

The Squads need to be inside the weapon range and if the Squad has moved, the Attack need to have the "Post movement" attribute.

Get Attacks:

Fields:

None

Requirement:

  • None

Description:

This script holds a list the AI's Squad attack ordered by their power. (Strongest attack is first)

Has Squad Attacked:

Fields:

None

Requirement:

  • None

Description:

This script returns true if the AI's Squad has attacked and false if it didn't.

Move Cursor To Attack Enemy:

Fields:

None

Requirement:

  • Enemy: Enemy Squad to attack.

Description:

This script will move the cursor on an enemy and will attack after. This script is will pause the rest of the AI until the attack is finished.

Move Toward Enemy:

Fields:

None

Requirement:

  • Enemy: Enemy Squad to move toward.

Description:

This script will move the AI's Squad within Attack range without attacking.

Move Toward Closest Enemy:

Fields:

None

Requirement:

  • None

Description:

This script will move the AI's Squad toward the closest Enemy.

End Squad Turn:

Fields:

None

Requirement:

  • None

Description:

This script will end that AI's Squad turn.
