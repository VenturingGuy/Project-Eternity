The Formula Parser will read an input and try to parse to compare values. The Core Formula Parser will parse data but can't interpret data on its own. The data acquisition must be implemented by another component such as the Deathmatch Formula Parser.

The Core Formula Parser handle all mathematical operation such as divisions, multiplications, modulos, addtions and subtractions and handle all comparaisons such as <, <=, >, >=, ==, !=, && and ||.

 

If the input starts with //C# then the input will be considered as C# code and the Roslyn compiler will be used.
