Counter:

Fields:

  • Starting value: Number to start counting from.
  • Increment value: Number to add to the Counter when the Increment Counter Trigger is used.
  • Ending value: Number at which the counter stop working and execute Counter Ended Event.

Triggers:

  • Increment Counter

Events:

  • Counter Updated
  • Counter Ended
