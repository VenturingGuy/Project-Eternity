Core AI

Core AI scripts are basic logic AI scripts that can be used on any Mod that support AI.

On Step:

Fields:

None

Requirement:

  • None

Description:

This script is called every time the AI is updated and call the linked scripts without restrictions.

Subroutine:

Fields:

Script to use path: Path to an AI file.

Requirement:

  • None

Description:

This script will execute another AI and will call the linked scripts when the selected AI returns with either the Return script or the Return Value script.

The Extract Return Value script can be used to retrieve values returned by this script.

Event:

Fields:

Script to use path: Path to an AI file.

Requirement:

  • None

Description:

Unlike the Subroutine, this script need to be called from another script, it works the same way as the On Step script.

 

This script will execute another and will call the linked scripts when the selected AI returns with either the Return script or the Return Value script.

The Extract Return Value script can be used to retrieve values returned by this AI script.

Return:

Fields:

None

Requirement:

  • None

Description:

This script returns null and is only used to end AI that will be used by Subroutine and Event scripts.

Return Value:

Fields:

None

Requirement:

  • None

Description:

This script returns an object and is only used to end AI that will be used by Subroutine and Event scripts.

You can return more than one object by using multiple Return Value scripts as they do not pause the AI.

Extract Return Value:

Fields:

  • Index: The index of which object to get. Starts at 0.

Requirement:

  • None

Description:

Retrieve an object returned from a Subroutine or Event script and send it to the linked scripts.

Logic Operator:

Fields:

  • Logic Operator: Which operator to use to compare the left object with the right object.

Requirement:

  • Left Number: A number that you want to compare.
  • Right Number: A number that you want to compare.

Description:

This script compared the value of the left number with the value of the right number using a logic operator. It will return true or false depending of the result of the comparison.

For Each:

Fields:

None

Requirement:

  • Object List: A list of object you want to loop through.

Description:

This script will call its following linked scripts one time for each object in the list used. It will also send the object it is curently using so it can be used directly with a Set Variable script.

If:

Fields:

None

Requirement:

  • Condition: A true or false value.

Description:

This script will call its following linked scripts based on the condition. If the condition is true it will only call the scripts linked through the "Condition is true" box. If the condition is false it will only call the scripts linked through the "Condition is false" box.

Set Variable:

Fields:

Default Value: An optional field that can be used so the script don't have to be initialized by another script.

Requirement:

  • None

Description:

Thie script is used to hold an object of any type. When called from an other script it will keep the object it received forever unless it is changed by being called again.

Get Content:

Fields:

None

Requirement:

  • Content: The object to get.

Description:

This script is used to send data from a data holder like the Set Variable script to its linked scripts. Since data holder can't execute other scripts, they can only send data through requirement boxes. This script use a requirement box to get the data then call the scripts that can't get data through requirement boxes.

For example, it can be used to get data from a Set Variable script and send it to another Set Variable script.

Get Random Content From List:

Fields:

None

Requirement:

  • Content: A list of objects.

Description:

This script works the same way as the Get Content script but only works with a list. It will return a random object contained in the list.

Get First Content From List:

Fields:

None

Requirement:

  • Content: A list of objects.

Description:

This script works the same way as the Get Content script but only works with a list. It will return the first object contained in the list.

Get List Count:

Fields:

None

Requirement:

  • Content: A list of objects.

Description:

This script returns the number of objects in a list. Can be used to know if a list is not empty.
