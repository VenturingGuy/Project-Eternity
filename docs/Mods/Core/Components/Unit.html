UNIT UPGRADES

All units can be upgraded via the Unit Upgrade menu. There are two upgrading systems: Individual and rank based. The system used is decided by a global setting: You can't have some units use one system and others use the other.

By default, the max number of upgrades is set at 10. In each unit's editor, you can set the max number of upgrades to be less than that: For example, a unit might have only 4 upgrades in mobility.

There is also an upgrade cap that can be changed midgame. This way it is possible to only allow units to be upgraded to 5 until the halfway point of the game and then increase it to 10.

The default upgrades are as follows:

  • HP: 200 per pip, 2000 total
  • EN: 10 per pip, 100 total
  • Armor: 60 per pip, 600 total
  • Mobility: 5 per pip, 50 total

INDIVIDUAL STAT UPGRADE SYSTEM

Each stat can be individually upgraded.In the unit editor, we can set the number of upgrades the unit can have, how much each upgrade costs, and how much an upgrade actually upgrades.

The default costs are as follows:

  • HP cost formula: 2000 + (2000 x Number of upgrades)
  • EN cost formula: 1000 + (1000 x Number of upgrades)
  • Armor cost formula: 2000 + (2000 x Number of upgrades)
  • Mobility cost formula: 2000 + (2000  x Number of upgrades)

RANK-BASED UPGRADE SYSTEM

A simplified version of the individual stat upgrade system. You upgrade all stats at once, including weapons: HP increases by 200, EN by 10, Armor by 60, Mobility by 5, Attacks by their upgrade rate.

This system is similar to SRC and will be easier to use for long time users or SRC.

ATTACK UPGRADES

When you upgrade a unit's attack, you upgrade all of them at once.

Each unit has an UPGRADE VALUE and an UPGRADE COST. The UPGRADE VALUE shows how much extra power the weapon gains when upgraded, COST shows how much the upgrade is going to cost.

Instead of giving each individual unit a separate VALUE and COST, they all select a RATE and a COST from a list: VERY SLOW, SLOW, NORMAL and FAST for RATE, CHEAPEST, CHEAP, NORMAL, EXPENSIVE for cost.

However, there is a SPECIAL CASE, those being MAP weapons: MAP weapons always upgrade by the VERY SLOW upgrade ranking.
