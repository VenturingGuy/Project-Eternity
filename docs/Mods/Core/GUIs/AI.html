AI Editor

The AI Editor allow to edit and AI.

An AI is made from  multiple AI scripts linked together, starting from a On Step script.

AI Preview

The AI preview panel lets you see, modify, link and move the AI scripts containted in the AI.

Drag and drap the AI scripts with the mouse to move them.

Hold the mouse inside one of the black boxes around an AI script to start the linking process. Left boxes can only be linked to boxes on the right side of AI scripts and bottom boxes can only be linked to boxes on the top of AI scripts.

The right and left boxes are execution links. An AI script will automatically launch, one at a time, every AI script linked to its right boxes in the order they are set. If an AI script is asynchronous, it will pause the rest of the AI and let the game run normally until it is finished. For instance an AI script that move and object could pause the rest of the AI until the object finish moving.

The top and bottom boxes are data links. AI scripts receive content from the boxes at the top and send content through the boxes at the bottom. There is no validation on the type of data sent and received so you have to make sure the AI script linked together are compatible. For instance, the "If" AI script only access true/false data, sending it a list of enemy will make it crash on execution.

 

AI Scripts

The list on the right contains every AI script usable by the Editor. Double clicking on one will add it to the AI preview section in the top left corner.

AI scripts are sorted by category, the Core category can be used in any kind of AI while others may be limited to specific mods. For instance the Deathmatch Scripts can only work inside the Deathmatch mod and will crash if used in another mod.

 

AI Properties

The box at the bottom right let you change the properties of some AI scripts.

It can be used to change the order of execution of the execution links.

It can also be used to add a comment that will be displayed in the bottom right of the script.

Some AI scripts like Set Variable have extra properties that can be set here. For instance the "Set Variable" can be set to have a default value.
