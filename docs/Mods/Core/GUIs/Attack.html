Attack Editor

The attack editor is used to create attacks Units can use in combat.

Item Information:

This section contains basic informations about the attack.

  • Name: The name of the attack.
  • Description: Flavor text if needed.
  • Edit Quotes: Decided which quoteset to use during combat. The quotesets are defined by the Character using that attack.

Stats:

  • Damage: The base damage the attack will do. If needed scripts can be used to change the damage depending of the combat.
  • EN cost: How much EN is used when using this Attack.
  • Maximum ammo: How many times the Attack can be used per map. Use 0 if there is no limit.
  • Morale requirement: How much morale the Pilot needs to use this Attack.
  • Minimum range: Define after how many tiles the Attack can hit.
  • Maximum range: Define the maximum distance in tiles the Attack can hit.
  • Accuracy: Bonus used for the Attack. Increase or decrease hit chances.
  • Critical: Bonus used for the Attack. Increase or decrease critical hit chances.

Secondary properties:

  • Post Movement: The attack can be used after moving.
  • Sword Cut: Not implemented
  • Shoot Down: Not implemented

Other Stats:

The rank affects accuracy and the final damage based on the location of the attacked target. If not set (value is '-'), the weapon can not be used on target from those location.

Bonus chart:

  • S: 20%
  • A: 10%
  • B: 0%
  • C: -15%
  • D: -30%

Attack type:

Decide if the accuracy and damage values are based on the pilot's MEL(Melee) stat or the RNG(Range) stat.

Types that uses MEL:

  • Blank
  • Melee
  • Solid blade
  • Energy blade

Types that uses RNG:

  • Solid shot
  • Energy shot
  • Remote
  • Special

Primary property:

Decide how you can attack other Unit.

  • None: Select a target on the map with a cursor, will only attack the leader of the targeted squad.
  • MULTI: Not implemented
  • BREAK: Not implemented
  • PLA: Same as none. If the Unit using having this attack is a wingman, it can use it to attack at the same time as the leader.
  • DASH: Not implemented. Move foward the target when attacking.
  • ALL: Same as none but will attack wingmans as well as the leader.
  • MAP: Select a target on the map using one a special pattern that can also hit allies. The attack can be used in 3 different ways:
    • Spread: Attack around the user.
    • Direction: Attack in a direction. (Left, Up, Right or Down)
    • Targeted: Attack around the cursor.

Pilot skills:

A list of skills that activate during combat.
